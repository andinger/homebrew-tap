# Homebrew Tap

Homebrew formulae for [andinger](https://github.com/andinger) projects.

## Install

```bash
brew tap andinger/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [teacher-toolkit-mcp](https://github.com/andinger/teacher-toolkit-mcp) | MCP server for teachers — material search, image research, and LehrplanPLUS integration |
| [vaultquery](https://github.com/andinger/vaultquery) | Query Obsidian vault files by YAML frontmatter |

### teacher-toolkit-mcp

```bash
brew install andinger/tap/teacher-toolkit-mcp
```

### vaultquery

```bash
brew install andinger/tap/vaultquery
```
